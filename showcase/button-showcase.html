Tab / Category Pills

dashboard tab strips, community section tabs, quick-action bars. Both .is-active and .is-selected trigger the orange active state — pick one and be consistent per page. Wrap them in .btn-tab-bar for correct spacing.

.btn--category.is-active .btn--category .btn-tab-bar (wrapper) 
<div class="btn-tab-bar">

  <button class="btn btn--category is-active">
    <i data-lucide="flag"></i> Report a Concern
  </button>

  <button class="btn btn--category">
    <i data-lucide="file-text"></i> Document Requests
  </button>

  <button class="btn btn--category">
    <i data-lucide="calendar"></i> Book Appointment
  </button>

</div>

<script>
  document.querySelectorAll('.btn--category').forEach(btn => {
    btn.addEventListener('click', function() {
      this.closest('.btn-tab-bar').querySelectorAll('.btn--category')
        .forEach(b => b.classList.remove('is-active'));
      this.classList.add('is-active');
    });
  });
</script>
Widget Buttons

Glass-style icon+label tiles for the time/alert widget overlay (dashboard home). Always used on dark/photo backgrounds. Icon stacks above the label.

.btn--widget 
<button class="btn btn--widget">
  <i data-lucide="shield"></i>
  <span>Docs</span>
</button>
Solid Buttons

Primary actions on white or light backgrounds. Green = confirm/register/official. Orange = submit/RSVP/report. Red = destructive/urgent.

.btn--green .btn--orange .btn--red 
<button class="btn btn--green">Register</button>
<button class="btn btn--orange">RSVP Now</button>
<button class="btn btn--red">Report Now</button>
<button class="btn btn--green" disabled>Disabled</button>

Loading State

Add .is-loading to any .btn to show the spinner. Use JS to toggle it. Works on all variants.

+.is-loading .btn__spinner 
<button class="btn btn--orange is-loading">
  <span class="btn__spinner"></span>
  Submitting...
</button>

<!-- Toggle with JS -->
btn.classList.add('is-loading');
btn.classList.remove('is-loading');

Filter Pills

Used to filter content lists on light backgrounds. Inactive = gray. Active = green. Not to be confused with .btn--category which is for dark backgrounds.

.btn--filter.is-active .btn--filter 
<button class="btn btn--filter is-active">All</button>
<button class="btn btn--filter">Announcements</button>

<script>
  document.querySelectorAll('.btn--filter').forEach(btn => {
    btn.addEventListener('click', function() {
      this.closest('div').querySelectorAll('.btn--filter')
        .forEach(b => b.classList.remove('is-active'));
      this.classList.add('is-active');
    });
  });
</script>
